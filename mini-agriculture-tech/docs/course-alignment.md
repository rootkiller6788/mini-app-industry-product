# Course Alignment — mini-agriculture-tech

## Nine-School Curriculum Mapping

### MIT
- **6.004 Computation Structures**: GPS coordinate systems, numerical computation (Haversine)
- **6.824 Distributed Systems**: Not directly (future: distributed farm sensor network)
- **IDS.131 Statistics**: WGEN stochastic weather generation, actuarial premium

### Stanford
- **CS 229 Machine Learning**: k-means yield zone clustering (agri_yield_map_zones)
- **EARTHSYS 155**: Carbon farming, IPCC GHG accounting
- **CS 448B Data Visualization**: Yield map spatial visualization

### Berkeley
- **ESPM 102D Climate & Energy**: Climate change impact on agriculture (RCP scenarios)
- **CS 267 HPC**: Future: GPU-accelerated field simulation
- **CE 107 Climate Change Mitigation**: Carbon sequestration modeling

### CMU
- **15-053 Optimization Methods**: Greedy crop allocation, rotation optimization
- **15-418 Parallel Computer Architecture**: Future: parallel field simulations
- **19-403 Sustainability**: Integrated assessment modeling

### UT Austin
- **CS 380D Distributed Systems**: Future: distributed farm IoT
- **CE 385D Water Resources**: Irrigation scheduling, water productivity

### ETH Zurich
- **263-3501 Parallel Programming**: Future: GPU kernel for simulation
- **701-1334-00**: Agri-environmental modeling (USLE, nitrogen cycle)

### Cambridge
- **Part II: Scientific Computing**: Numerical methods (Penman-Monteith, integration)
- **Part II: Environmental Engineering**: Water quality, nutrient runoff

### Tsinghua
- **操作系统**: System programming (make, build, library structure)
- **农业工程**: Precision agriculture technology, variable rate application

### Georgia Tech
- **CS 7641 Machine Learning**: Future: ML-based pest prediction
- **CP 6514 Urban Ecological Design**: Green infrastructure, carbon farming
